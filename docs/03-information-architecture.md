## 3. Sections to Avoid
- Fake testimonial carousel
- Inflated statistics block
- Heavy embedded third-party widgets
- Long founder biography blocks with no conversion function
- Duplicate WhatsApp join button on-page

## 4. Final IA Summary
Use a single-page conversion sequence that answers four user questions in order: What is this? Is it for me? Is it credible and practical? How do I register now? Keep copy concrete, visual rhythm clean, and CTAs consistently available.
