# 1. Product Summary
Lumenet Labs Presents: Evolve with AI is a premium, conversion-focused, student-first landing experience for a free 8-day live online workshop running from 1 July to 8 July 2026, 7:30 PM to 9:30 PM IST. The site explains practical AI applications for real life and drives registration to a Google Form.

# 2. Primary Goal
Maximize qualified registrations to [Google Form](https://forms.gle/oViwgzHnwdGuMUZK9) with low-friction CTAs and clear event value.

# 3. Secondary Goals
- Build trust through transparent event facts and practical curriculum.
- Improve clarity for first-time AI learners.
- Position Lumenet Labs as practical and ethical.
- Support easy mobile-first scanning for students and parents.

# 4. Target Audience
- High school students
- College applicants
- University students
- Early professionals
- Creators and builders
- Founders and business-minded learners
- Underserved learners seeking free practical opportunity

# 5. Core User Pain Point
Users know AI exists but do not know how to apply it reliably for study, applications, career decisions, productivity, and practical project outcomes.

# 6. Core Transformation
From random AI usage to structured, ethical, reusable AI workflows that improve learning, execution speed, and life productivity.

# 7. Key Offer Details
- Event: Lumenet Labs Presents: Evolve with AI
- Date: 1 July to 8 July 2026
- Time: 7:30 PM to 9:30 PM IST
- Duration: 8 days, 16 total hours
- Format: Live online via Google Meet
- Cost: Completely free
- Resources: Worksheets, templates, practical prompts, guided resources
- Certificates: Available for active participants
- Registration: [https://forms.gle/oViwgzHnwdGuMUZK9](https://forms.gle/oViwgzHnwdGuMUZK9)
- Replay policy: No replay promise

# 8. Brand and Visual Direction
Premium dark base with metallic gold hierarchy and restrained cyan AI energy accents. The page should feel cinematic, elegant, sharp, trustworthy, and readable.

# 9. Website Structure
Single landing page plus branded custom `404.html`, optimized for registration conversion.

# 10. Registration Funnel
- Hero CTA (`Register Free`)
- Section-level repeated CTAs
- Registration section with trust microcopy
- Final CTA block
- All registration actions open Google Form in new tab

# 11. Technical Stack
- Plain HTML
- Modern CSS with custom properties
- Vanilla JavaScript (no frameworks)
- Free Google Fonts
- Netlify static hosting

# 12. Accessibility Requirements
- Semantic structure and heading hierarchy
- Skip link
- Keyboard-accessible nav/menu/FAQ/modal
- Visible focus styles
- Reduced motion support
- Mobile touch-friendly targets
- High contrast text on dark background

# 13. Performance Requirements
- Static-first architecture
- Deferred JavaScript
- Minimal dependencies
- Optimized media
- Mobile Lighthouse target 90+

# 14. Privacy, Cookies, and Analytics
- Minimal collection on website itself
- Registration handled via Google Forms
- Cookie banner with accept/decline using localStorage
- Optional analytics only after consent

# 15. Deployment Recommendation
Primary platform: Netlify (free tier), GitHub-connected static deploy, HTTPS, cache headers, and custom 404 routing.

# 16. Launch Priorities
1. Accurate content and event details.
2. Mobile-first readability and CTA clarity.
3. Accessibility basics and keyboard flow.
4. Fast load and stable visual quality.
5. Metadata and social preview readiness.

# 17. Non-Negotiable Decisions
- Free/open-source stack only.
- Static site in HTML/CSS/JS.
- Netlify primary deploy.
- Homepage + custom 404 required.
- Dark premium black/gold/cyan identity.
- No fake social proof.
- No replay promise.
- Certificates phrased as active-participant only.
- Existing posters with outdated time/date excluded from homepage UI.
